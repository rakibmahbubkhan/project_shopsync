# Project_ShopSync

Modern Inventory Management System

## Stack

- Laravel 12 (API)
- Vue 3 (SPA)
- Tailwind CSS
- MySQL

## Installation

### Backend

cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate

### Frontend

cd frontend
npm install
npm run dev
